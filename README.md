## Gerenciador de Biblioteca Pessoal

Aluno: Willian Pontieri Shibata 

## Descrição do projeto

O projeto é um sistema de gerenciamento de biblioteca pessoal desenvolvido em TypeScript.
Permite cadastrar/remover livros, marcar leituras, buscar por autor ou título e gerar estatísticas da coleção.

## Tecnologias

- Node.js 18+
- TypeScript 6+
- Arrays tipados para armazenamento de dados

## Instalação

```bash
# Instalar dependências
npm install

# Compilar TypeScript
npm run build

# Modo desenvolvimento (watch mode)
npm run dev
```

##  Execução

```bash
npm run dev
```

##  Funcionalidades

- Cadastro e remoção de livros
- Busca por título e autor
- Marcar livros como lidos com avaliação
- Estatísticas da biblioteca
- Classificação por década
